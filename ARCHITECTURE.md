## 10. 配置示例

### 数据源优先级配置 (动态优先级)

```python
# data_provider/qveris_fetcher.py - QVerisFetcher

class QVerisFetcher(BaseFetcher):
    """动态优先级数据源"""
    
    priority = 2  # 默认优先级（股票）
    CRYPTO_PRIORITY = 0  # 加密货币主数据源
    STOCK_PRIORITY = 2   # 股票备用数据源（等同于Tushare）
    
    def get_priority_for_code(self, stock_code: str) -> int:
        """根据代码类型返回动态优先级"""
        if _is_crypto_code(stock_code):
            return self.CRYPTO_PRIORITY  # 加密货币: Priority 0
        return self.STOCK_PRIORITY       # 股票: Priority 2
```

```python
# data_provider/base.py - DataFetcherManager

# 数据源列表（按优先级排序）
self._fetchers = [
    efinance,     # Priority 0 (免费, 最优先)
    akshare,      # Priority 1 (免费)
    crypto,       # Priority 1 (免费, 加密货币专用)
    pytdx,        # Priority 2 (免费)
    tushare,      # Priority 2 (收费)
    qveris,       # Priority 2 (股票) / Priority 0 (加密货币主数据源)
    baostock,     # Priority 3 (收费)
    yfinance,     # Priority 4 (兜底)
]
]
```
